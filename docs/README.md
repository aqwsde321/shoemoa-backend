# Shoemoa Backend Documentation

이 디렉토리는 Shoemoa 백엔드 프로젝트의 아키텍처, 구현 상세, 테스트 전략 및 문제 해결 기록을 포함합니다.

## 📂 Directory Structure

### 🏗️ architecture
프로젝트의 핵심 설계 원칙과 구조를 설명합니다.
- **CORE.md**: 비즈니스 로직 처리 원칙 (Entity vs Service)
- **STRUCTURE.md**: 패키지 및 코드 작성 템플릿
- **CONVENTIONS.md**: 네이밍 규칙 및 코딩 컨벤션
- **old/**: 이전 버전(v0~v2)의 아키텍처 문서 아카이브

### 🧪 testing
테스트 작성 가이드와 전략을 다룹니다.
- **TESTING.md**: Domain Test 및 Integration Test 작성 방법

### ⚙️ implementation
특정 기능이나 인프라의 구체적인 구현 방법을 설명합니다.
- **README.md**: 구현 상세 문서 목록 및 설명

### 🔧 troubleshooting
개발 중 발생한 문제와 해결 과정을 기록합니다.
- **README.md**: 트러블슈팅 목록 및 작성 가이드
