# Architecture Archives

이 디렉토리는 Shoemoa 백엔드 프로젝트의 이전 버전 아키텍처 문서들을 보관합니다.

프로젝트가 발전하면서 변경된 설계 사상이나 구조의 히스토리를 파악하는 데 사용할 수 있습니다.

- **v0**: 프로젝트 초기 설계 원칙을 다룹니다. "도메인 = JPA Entity"라는 기본 원칙과 레이어별 책임 분리를 정의합니다.

- **v1**: v0의 설계를 구체화하여 **"Port를 통한 Aggregate 간 협력"** 모델을 처음 도입하고, 3가지 핵심 규칙(로직은 Entity에, Service는 흐름만, Port로 협력)을 명시한 버전입니다.

- **v2**: v1의 설계 원칙을 그대로 계승하며, 용어사전(`GLOSSARY.md`) 링크를 추가하는 등 **문서의 완성도를 높인 버전**입니다. 설계 사상의 변경은 없습니다.
