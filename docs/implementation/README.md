# Implementation Details

이 디렉토리는 Shoemoa 백엔드 프로젝트의 특정 기능 또는 인프라 구현에 대한 상세 문서를 포함합니다.

## 📂 Directory Structure

### 🔒 security
인증(Authentication) 및 인가(Authorization)와 같은 보안 관련 구현 상세를 다룹니다.
- **JWT_AUTHENTICATION.md**: JWT 기반 인증 및 인가 시스템 설명

### ⚙️ infrastructure
AWS S3, CloudFront CDN 등 프로젝트에서 사용하는 인프라 관련 설정 및 구현 방법을 설명합니다.
- **AWS_S3_CDN_SETTING.md**: AWS S3와 CloudFront를 사용한 이미지 업로드 및 CDN 배포 설정 가이드
